# 📈 MesclaInvest

> **Projeto Integrador 3** | Engenharia de Software – **PUC-Campinas**

Aplicação mobile desenvolvida em **Flutter** focada em organização, escalabilidade e alto padrão de desenvolvimento. O **MesclaInvest** simula uma plataforma de investimentos voltada para startups do ecossistema de inovação **Mescla**.

---

## 📌 Sobre o Projeto

O sistema permite a simulação de compra e venda de tokens que representam participações em startups vinculadas à universidade. O objetivo é puramente acadêmico e educacional, sem o envolvimento de ativos financeiros reais.

### 🚀 Funcionalidades Principais
* **Gestão de Conta:** Criação de perfil e autenticação segura.
* **Marketplace de Startups:** Visualização e consulta de informações institucionais e societárias.
* **Interação:** Envio de perguntas diretamente aos empreendedores.
* **Simulação de Investimentos:** Compra e venda de tokens de participação.
* **Dashboards:** Acompanhamento gráfico da valorização dos ativos.
* **Documentação:** Acesso a atualizações e conteúdos públicos das empresas.
